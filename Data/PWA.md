# Progressive Web App(PWA)

**Web 網頁和 App 的混合體**，使網頁具有 App 應用程式般的使用經驗。PWA 使用HTML標籤、CSS樣式及javascript 程式控制功能所製作的網頁，可具有 Responsive Web Design (RWD) 性能，混合 App 與網頁的特性驗

- 可以進行離線操作
- 可以下載且安裝到桌面
- 沒有URL列，可以全螢幕操作
- 具備有推播 Push通知的功能
- 不需經由 App store，就可以安裝到手機、平版主畫面或是桌上型電腦的桌面
